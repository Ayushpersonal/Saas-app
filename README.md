# 🚀 SaaS Nexus

Welcome to **SaaS Nexus**, a modern Software-as-a-Service platform built with performance, scalability, and premium aesthetics in mind.

## 🛠 Tech Stack

- **Frontend:** React.js + Vite
- **Routing:** React Router 7
- **Backend:** Node.js + Express
- **Styling:** Premium Vanilla CSS (Modern CSS features)
- **Database:** (To be determined)

## 🏗 Project Architecture

```text
src/
 ├── components/     // reusable UI (cards, buttons, sidebar)
 ├── pages/          // full pages (Dashboard, Users,Settings)
 ├── layouts/        // layout wrappers (Sidebar + Navbar)
 ├── hooks/          // custom hooks
 ├── services/       // API logic (later)
 ├── utils/          // helpers
 ├── assets/
 ├── App.jsx
 └── main.jsx
```
